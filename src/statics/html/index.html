Redis Caching Example

What is this?

This is a display of speed differences between standard database access and cached access.

How does it work?

Use one of the following commands to set or get variables (without the brackets)

  • SET {type} {name} {value}
  • GET {name}
  • DEL {name}

Where:

  • {name} is the variable name
  • {type} is either int or string
  • {value} is
    • an integer for int types (e.g. -13, 55, 150)
    • a sequence of characters surrounded by quotes for string types (e.g. "hi", "hello, world!")
variables expire from the cache after 15 seconds
